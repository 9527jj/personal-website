# Personal Website

这是一个静态个人主页，可以直接部署到 GitHub Pages、Netlify 或 Vercel。

## GitHub Pages 发布步骤

1. 打开 GitHub，新建一个公开仓库，例如 `personal-website`。
2. 上传本文件夹里的所有文件，包括 `index.html`、`styles.css`、`.nojekyll` 和 `assets` 文件夹。
3. 进入仓库的 `Settings` -> `Pages`。
4. 在 `Build and deployment` 里选择 `Deploy from a branch`。
5. Branch 选择 `main`，文件夹选择 `/root`，保存。
6. 等待 1-3 分钟，GitHub 会生成公开网址。

发布后，把邮箱、姓名、项目介绍和页面标题替换成你的真实信息。
